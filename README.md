# ipfs-manager
A C/S based GUI tool for manage files in ipfs or ipfs private network.

## Server Deployment

Written in Django.
in Project server directory:
```shell
python3 manage.py makemigrations
python3 manage.py migrate
```

## Client Useage

Writen in PyQt5

in Project **root directory** :
```shell
python3  client.py
```
